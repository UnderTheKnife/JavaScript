  12. Дан массив [1, 2, 3, 4, 5]. С помощью метода splice сделайте из него массив [1, 'a', 'b', 2, 3, 4, 'c', 5, 'e'].

  13. Дан массив [3, 4, 1, 2, 7]. Отсортируйте его.
